Subscribe to a set

Text "subscribe (set)", with either the set name or the ID number, to subscribe to a set. Brick Hound sniffs the most popular sets right now, so give it a try!

Receive alerts

When Brick Hound smells a change, it'll text you right away with a link to the set. You're guaranteed to be the first to know when a set comes in stock!

List

Keep track of sets that you're tracking by texting "list".

Unsubscribe

Finally got the set you were looking for? Simply text "unsubscribe (set)" so Brick Hound knows to stop barking at you about it.

By opting into Brick Hound, you authorize Brick Hound to store your phone number and send you text messages. Your phone number will only be used to send alerts as requested by you. No other personally identifying information will be collected by Brick Hound, and your phone number will not be shared. You may text "STOP" at any time to immediately terminate all text messages - service will only resume if you type "START".
